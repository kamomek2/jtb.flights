## JTB Flights Component

Will dispatch the search and handle routing.

### Setup
Clone and install:

```
git clone https://github.com/muerwre/jtb.flights.git
cd jtb.flights
yarn
```

Run it:
```
yarn start
```

This component can be launched as standalone app, see it in action by navigating to 
```http://localhost:8001/```.
